This folder contains sequence alignments in fasta format for the Cox1 and Mp20 loci used in the study of Papadopoulou & Knowles (2015): "Species-specific responses to island connectivity cycles: refined models for testing phylogeographic concordance across a Mediterranean Pleistocene Aggregate Island complex"


Sequence names contain a six-digit voucher specimen code (see supplementary material) followed by a 4-letter indicating the species name and the Island where the sample comes from. See abbreviations below:

In the case of the nuclear gene Mp20, the sequences have been phased using the PHASE algorithm, and the two haplotypes are presented separately (and marked as 1 or 2, see the last digit of sequence name).
